# 原型链

```javascript
function Person() {}

// 虽然写在注释⾥，但是你要注意：
// prototype是函数才会有的属性
Person.prototype.name = "Naparte";
var person1 = new Person();
var person2 = new Person();
console.log(person1.name); // Naparte
console.log(person2.name); // Naparte
```
